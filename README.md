ASSIGNMENT 4 Module 5: Files, Exceptions, and Errors in Python

  ----------
  Overview
  ----------

This assignment contains two tasks related to file handling and
exception handling in Python.

Task 1 focuses on reading a file and handling errors. Task 2 focuses on
writing and appending data to a file.

  ---------------------------------------
  Task 1: Read a File and Handle Errors
  ---------------------------------------

Problem Statement:

Create a Python program that:

1.  Opens and reads a text file named sample.txt
2.  Prints the content line by line
3.  Handles errors properly if the file does not exist

Expected Output:

If the file exists: The program should display the content of sample.txt
line by line.

If the file does not exist: The program should display an appropriate
error message indicating that the file was not found.

  -----------------------------------------
  Task 2: Write and Append Data to a File
  -----------------------------------------

Problem Statement:

Create a Python program that:

1.  Takes input from the user
2.  Writes the input into a file named output.txt
3.  Appends additional information to the same file
4.  Reads and displays the final content of the file

Expected Output:

For example, if the user enters 25:

The program should store the value in the file, append additional data,
and then display the final content of the file.
